# 📚 Book Data Filtering Project

A simple JavaScript project demonstrating the use of the `Array.filter()` method to extract data from a dataset.

## 🔍 Features

- Fetch all History genre books
- Fetch Fictional books published after 1995
- Clean and modular code structure
- Beginner-friendly

## 🛠 Technologies Used

- JavaScript (ES6)
- Node.js (optional for execution)

## ▶️ How to Run

1. Clone the repository
2. Open terminal in project folder
3. Run:

   node index.js

## 📖 Learning Outcome

- Understanding of Array.filter()
- Conditional data extraction
- Writing modular and readable JavaScript code
